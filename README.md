# scala-cli-millbuild-juni-hello-world

## Description
A POC for mill with junit
and scala.

## Tech stack
- scala
- junit
- mill

## Docker stack
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`
- test results
  - console
  - file
    - bin/out/HelloWorld/test/test/meta.json

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
